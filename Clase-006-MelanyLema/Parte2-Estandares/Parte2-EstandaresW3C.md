# Parte 2: Exploración y selección de estándares W3C
A continuación, presento la selección de 5 estándares fundamentales del W3C, junto con su descripción y la justificación de por qué son esenciales en el desarrollo web moderno.

## 1. HTML5 (HyperText Markup Language)
### Descripción: 
Es la quinta revisión y el estándar central de la World Wide Web. Define la estructura semántica y el contenido de las páginas web, introduciendo nuevas etiquetas que aportan significado al contenido (como nav, article, section) y soporte nativo para multimedia.

**Justificación:** Lo elegí porque es el esqueleto indiscutible de cualquier aplicación web. Sin HTML5 y su semántica, los navegadores y motores de búsqueda no podrían interpretar la jerarquía ni la lógica de la información, haciendo imposible la interoperabilidad.

## 2. CSS3 (Cascading Style Sheets)
### Descripción: 
Es el lenguaje utilizado para describir la presentación visual de un documento estructurado en HTML. Esta versión modular permite controlar colores, fuentes, animaciones y, lo más importante, el diseño adaptativo (Responsive Design) mediante Media Queries.

**Justificación:** Es fundamental para la Experiencia de Usuario (UX). Permite la separación de preocupaciones (contenido vs. diseño), lo que facilita el mantenimiento del código y asegura que la web se vea bien tanto en un celular como en una pantalla gigante.

## 3. WCAG (Web Content Accessibility Guidelines)
### Descripción: 
Son documentos técnicos que explican cómo hacer que el contenido web sea accesible para personas con discapacidades (visuales, auditivas, motoras o cognitivas). Se basan en cuatro principios: Perceptible, Operable, Comprensible y Robusto.

**Justificación:** La web debe ser universal. Elegí este estándar porque representa el componente ético y social de la ingeniería de sistemas, garantizando que nadie sea excluido del acceso a la información por limitaciones físicas.

## 4. SVG (Scalable Vector Graphics)
### Descripción: 
Es un formato de gráficos vectoriales bidimensionales basado en XML. A diferencia de las imágenes de mapa de bits (JPG o PNG), los gráficos SVG se definen mediante matemáticas, lo que permite escalarlos infinitamente sin perder calidad.

**Justificación:** Es vital en la web actual debido a la variedad de resoluciones de pantalla (Retina displays, 4K). Permite crear íconos, logotipos e ilustraciones que cargan rápido y se ven nítidos en cualquier dispositivo.

## 5. JSON-LD (Linked Data)
### Descripción: 
Es un estándar para codificar datos vinculados (Linked Data) utilizando JSON. Permite estructurar la data dentro de una página web para que las máquinas (como los bots de Google) entiendan el contexto del contenido (ej. saber si un bloque de texto es una receta, un evento o un producto).

**Justificación:** Lo seleccioné por su importancia en el SEO moderno y la web semántica. Ayuda a conectar la información de manera que las aplicaciones inteligentes puedan procesarla automáticamente, mejorando la visibilidad del sitio web.
